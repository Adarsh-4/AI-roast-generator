# ⚡ AI Roast Generator

A futuristic **AI-style Roast Generator Web App** built using **HTML, CSS, and JavaScript**.
This project generates funny **Hindi roasts** for any name entered by the user in a **ChatGPT-style interface** with a modern **glassmorphism UI** and **neon animated background**.

---

## 🚀 Features

* 🤖 ChatGPT-style roast chat interface
* 🎨 Glassmorphism futuristic UI
* 🌌 Neon animated particle background
* 🔥 Savage meter showing roast intensity
* 🇮🇳 Hindi roast dataset (100+ roasts)
* ⌨️ Typing animation for AI response
* 📋 Copy roast to clipboard
* 📱 Share roast directly to WhatsApp
* 📱 Fully responsive design
* ⏳ Animated loading screen

---

## 🛠 Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Canvas API (for animated particles)

## ⚙️ How to Run the Project

1. Download or clone the repository.

2. Open the project folder.

3. Double-click **index.html**
   or open it in any web browser.

4. Enter a name and click **Roast**.

The AI will generate a funny Hindi roast instantly.
